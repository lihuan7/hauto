При разработке использованы следующие технологии:

Java 8, JSP/Servlet, JQuery, AJAX.
СУБД – Oracle/PostgreSQL,
контейнер сервлетов – Tomcat,
сборка проекта – Maven.

Так как не было указано каким именно способом организовать доступ
к БД посчитал оптимальным использовать ORM Hibernate в реализации JPA.

Для создания и заполнения БД можно использовать скрипт AppTest для
этого необходимо в persistence.xml произвести настройки БД и переключить
в режим работы create-drop.

Так же в катологе prepare-server файлы конфигурации tomcat которые необходимо
скопировать в  папку tomcat/conf.
Там же в папке sql находится полный дамп бд  hauto  - hauto_db_dump.sql.
